This profile contains practical and experimental projects involving:

- industrial and HomeLab infrastructure;
- Linux, LXD, Docker and Portainer;
- network visibility and troubleshooting;
- Python and PowerShell automation;
- monitoring dashboards and data visualization;
- OT protocols and integration concepts;
- lightweight tools designed for restricted or low-resource environments.

- real HomeLab topology may be used for educational examples;
- network tools are intended only for systems the operator owns or is authorized to test.
