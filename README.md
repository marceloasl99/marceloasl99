# Featured Projects

## OT, Industrial Networking, and Diagnostics

### [Cisco C9200 OT Investigation Collector](https://github.com/marceloasl99/cisco-c9200-ot-investigation-collector)

A time-bounded, read-only PowerShell collector for Cisco Catalyst 9200 and IOS-XE investigations in OT networks.

Key capabilities:

- multi-switch collection through PuTTY Plink;
- uptime, hardware, stack, CPU, memory, environment, and software data;
- interface state, traffic, counters, errors, optics, and err-disable analysis;
- VLAN, trunk, EtherChannel, STP, MAC, ARP, CDP, and LLDP collection;
- port security, authentication, ACL, DHCP Snooping, DAI, IP Source Guard, and storm-control visibility;
- optional focused VLAN and MAC investigations;
- hard timeout per switch with partial-output preservation;
- one consolidated timestamped TXT report.

**Technologies:** PowerShell · Cisco IOS-XE · Catalyst 9200 · Plink · SSH · OT Networking

---

### [Modbus TCP Server Monitor](https://github.com/marceloasl99/modbus-tcp-server-monitor)

A Windows PowerShell Modbus TCP server simulator with a responsive real-time monitoring GUI.

Key capabilities:

- multiple concurrent Modbus TCP clients;
- real-time RX and TX frame visualization;
- packet and byte counters per client;
- active-client monitoring;
- address and value-change tracking;
- persistent holding-register and coil values;
- complete session logs and separate value-change logs;
- support for FC01, FC02, FC03, FC04, FC05, FC06, FC15, and FC16.

**Technologies:** PowerShell · Windows Forms · C# · Modbus TCP · Industrial Automation

---

### [Port Scanner](https://github.com/marceloasl99/portscanner)

An experimental TCP service-checking toolkit for authorized HomeLab and local-network environments.

Key capabilities:

- checks one or multiple IPv4 targets;
- presets for common, web, remote-access, Windows, database, and industrial/IoT ports;
- clear per-target results and connection timing;
- PHP server-side TCP checks;
- local Windows inspection mode for listening ports, processes, PIDs, and exposure scope;
- copy-ready reports.

**Technologies:** PowerShell · PHP · HTML · CSS · JavaScript · TCP/IP

---

### [Windows Full System Report](https://github.com/marceloasl99/windows-full-system-report)

A comprehensive PowerShell inventory and diagnostic report generator for Windows 10 and Windows 11.

Coverage includes:

- hardware, firmware, storage, memory, GPU, USB, and PnP devices;
- IP, DNS, routes, Wi-Fi, interfaces, listeners, and active connections;
- software, services, processes, startup entries, and scheduled tasks;
- Defender, firewall, Secure Boot, TPM, BitLocker, and security policies;
- users, groups, sessions, shares, printers, updates, virtualization, and event diagnostics;
- one timestamped UTF-8 TXT report.

**Technologies:** PowerShell · Windows · WMI/CIM · Networking · Security Diagnostics
